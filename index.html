About Block-Ads

Block-Ads provides an advertisment-free and distraction-free Internet experience for everyone. We are providing our services in USA, Europe, Russia and Africa on Chrome & Firefox browser. It is designed to give users control over what they see in their web browser. It is a software capability for blocking or altering online advertising in a web browser or a network. This is done by using our browser extensions.

What is Our Motive?

  • Protecting privacy.
  • Reduces the number of HTTP cookies, browser fingerprinting and other aggressive behavioral tracking techniques.
  • Protecting from malvertising.
  • Saving bandwidth.
  • Better user experience.

How we block ads?

While a site is loading, the Block-Ads software checks the domain names of the elements loading on the web page against massive blacklists. If any site component is flagged as advertising material, the Block-Ads stops it from loading. It will block commercials from every media player in your browser. After the page content is displayed, the ad blocker looks for elements that meet certain rules, e.g. images in common banner dimensions, and hides them from the screen.

Is using Block-Ads legal?

Yes, it’s perfectly legal to use Block-ads for blocking advertisments. While websites will specifically curate what they show or advertise to consumers, it’s up to user behavior to determine how they want to consume that content. It just so happens that the perfect user experience is without ads.
